# Global 2D GIA Model: FastEarth

## Background
FastIsostasy is a regional Glacial Isostatic Adjustment (GIA) model that solves the
governing equations in 2D. It uses a viscosity layer-stacking method to compress a
3D Earth viscosity field into an effective 2D spatially variable field.

The original description of FastIsostasy can be found here: https://gmd.copernicus.org/articles/17/5263/2024/gmd-17-5263-2024.xml__

## Goal
Develop a global 2D GIA model using the same layer-stacking approach. The model takes
a 2D spatially variable Earth viscosity field as input.

## Tasks
1. Define the governing equations for the global 2D model.
2. Outline a numerical strategy to solve them.
