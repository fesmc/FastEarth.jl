# FastEarth

[![Docs](https://img.shields.io/badge/docs-stable-blue.svg)](https://fesmc.github.io/FastEarth.jl/)

Global 2D Glacial Isostatic Adjustment (GIA) model, generalising
[FastIsostasy](https://gmd.copernicus.org/articles/17/5263/2024/) to the sphere.

The canonical specification lives in the rendered documentation:
**https://fesmc.github.io/FastEarth.jl/**

Source for the docs is under [`docs/src/`](docs/src/). Edits to the design
should be made there, not in any other copy.
